HTTP 响应模板配置

配置 TeamServer 返回给 Beacon 的 HTTP 响应伪装模板

模板工作原理

TeamServer 会将加密的任务数据嵌入到 HTTP 响应体中返回给 Beacon。模板中的 {{DATA}} 占位符会在运行时被替换为实际的加密数据。修改模板后请确保 Beacon 端的解析逻辑与之匹配。

请确保模板中包含且仅包含一处 {{DATA}} 占位符

端点轮转工作原理

当 Beacon 与当前 C2 端点连续通信失败达到阈值后，会自动切换到下一个备用端点继续通信。配置保存后，需要通过「推送到 Beacon」将端点列表下发到指定 Beacon。
